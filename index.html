🪡 Welcome to LeatherWorks

This app uses Google Sheets as your database — so all your data syncs across every device automatically. You only need to do this once.

  1. Open Google Sheets and create a new spreadsheet
  2. Name it "LeatherWorks Data"
  3. Go to Extensions → Apps Script, paste the API script (see below), and deploy it as a Web App
  4. Copy the Web App URL and paste it below
💡 For now, the app works in offline demo mode with sample data. You can connect your Google Sheet at any time in Settings.
